1) Зарегистрироваться/авторизоватся на gogs.wsr.ru:3000
2) Создать репозиторий, внизу убрать галочку "Инициализировать этот репозиторий..."
3) Открыть Git Bash
4) Используя команду cd, перейти в каталог с файлами, которые надо загрузить в репу (нр: User05)
5) git init - создает локальный репозиторий
6) git config --global user.email "email, указанный в профиле gogs"
	      replace-all user.email "новый email для замены"
7) git config --global user.name "имя пользователя, указанное в профиле gogs"
	      replace-all user.name "новый имя пользователя для замены"
8) git checkout -b <название ветки> - создание новой ветки

git status - посмотреть какие файлы отслеживаются

9) git add <имя файла или папки> - добавляет в коммит указанные файлы/директории
10)git commit -m <название коммита> - создает коммит с указанными файлами
11)git remote add <название ветки> <путь к репозиторию (на сайте gogs)>
12)git push --all <имя ветки>

Повторить действия 8-12 для разных веток
